# explain-flags

Explain sam flags according to [picard](https://broadinstitute.github.io/picard/explain-flags.html)

## How to use

**install**

```shell
cargo install --git https://github.com/kwuiee/explain-flags
```

**usage**

```shell
$ ef 147
[✓] read paired
[✓] read mapped in proper pair
[✓] read reverse strand
[✓] second in pair
```
